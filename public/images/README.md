# Imágenes del Portfolio

Coloca las imágenes en esta carpeta (`public/images/`) con los siguientes nombres exactos:

## Página Home (Portada)
- **home-hero.jpg** - Imagen principal de la portada
  - Formato: Horizontal/Landscape
  - Tamaño recomendado: Mínimo 1600px de ancho

## Página Work

### Primera imagen (derecha, pequeña)
- **work-image-1.jpg** - Primera imagen que aparece a la derecha
  - Formato: Vertical/Portrait (proporción 3:4)
  - Tamaño recomendado: Mínimo 800px de ancho

### Segunda imagen (izquierda, mediana)
- **work-image-2.jpg** - Segunda imagen que aparece a la izquierda
  - Formato: Vertical/Portrait (proporción 3:4)
  - Tamaño recomendado: Mínimo 900px de ancho

### Tercera imagen (borde izquierdo en el centro)
- **work-image-3.jpg** - Tercera imagen con borde izquierdo alineado al centro
  - Formato: Vertical/Portrait (proporción 3:4)
  - Tamaño recomendado: Mínimo 800px de ancho

## Formatos soportados
- **JPG** (recomendado)
- PNG
- WebP

## Instrucciones
1. Descarga o prepara tus imágenes
2. Renombra los archivos exactamente como se indica arriba
3. Coloca todos los archivos en esta carpeta: `public/images/`
4. Los cambios se reflejarán automáticamente en la web

## Nota importante
- Los nombres de los archivos deben ser **exactos** (con la extensión .jpg)
- Si usas otro formato (PNG), cambia la extensión en el código también
- Las imágenes en la carpeta `public/` se sirven directamente desde la raíz del sitio
